# SqlUNet resources

* **sqlunet** for SQL-unified *-nets:
	*  WordNet
	*  Open English Wordnet
	*  SyntagNet
	*  Verbnet
	*  SemLink
	*  PropBank
	*  Framenet
	*  PredicateMatrix
	*  SUMO
	*  British National Corpus
	
* **semantikos** for Android semantikos applications, derived from sqlunet
* **wndb** for WordNet databases in WNDB format
